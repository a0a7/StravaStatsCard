## Configuration Options:
### Sport - Cycling/Running
Supports statistics from both cycling/running.

### Units - SI/Imperial
Suports both kilometers/meters and miles/feet.

### Color Scheme
Supports light/dim/dark color schemes.

### Profile Flag (Optional)
Takes user input of any country, territory, or American state name and prints a small flag icon on the card.

Uses the https://countryflags.com API.
